## System Requirements

Unity 2018.3.0 or later versions.<br />

## Controls

**Keyboard And Mouse**
- Movement: **WASD**
- Camera: **Mouse**
- Jump: **Space**
- Toggle Walk/Run: **Caps Lock**
- Sprint: **Left Shift**
- Toggle Slow Motion: **Tab**

**XBOX 360/ONE Controller**
- Movement: **Left Stick**
- Camera: **Right Stick**
- Jump: **A**
- Sprint: **LB**
- Toggle Slow Motion: **RB**
